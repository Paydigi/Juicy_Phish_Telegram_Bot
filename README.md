# Juicy_Phish_Telegram_Bot

Self hosted Telegram Bot that runs Locally & loves to Explore. And sometimes even brings some Treasures back.

Go to - https://skerritt.blog , choose AI model. Once you find a model you like, download the .bin onto your computer.

For Mac OS / Linux
$ git clone git@github.com:LostRuins/koboldcpp.git
$ cd koboldcpp
$ make

$ wget https://huggingface.co/localmodels/WizardLM-7B-Uncensored-4bit/resolve/main/ggml/wizardlm-7b-uncensored-ggml-q4_1.bin
$ python3 koboldcpp.py /home/autumn/models/wizardlm-7b-uncensored-ggml-q4_1.bin.1 9057
